Industrial Robot Maze Navigation

A vision-guided robotic manipulation system enabling an industrial robot to autonomously navigate a complex 3D maze using calibrated perception, custom kinematics, and trajectory planning.

OVERVIEW

The project involved designing a complete perception–control pipeline enabling a Mitsubishi RV-6S industrial robot to autonomously navigate a 3D rod-shaped maze using a ring-shaped end-effector. The system used an overhead Basler RGB camera for pose estimation and custom trajectory planning to follow complex spatial paths without collisions.

The robot performs intrinsic camera calibration using ChArUco markers, detects maze pose with ArUco, aligns its end-effector, and executes the required variant trajectory (A–D). For orientation-dependent segments, a custom IK-selection strategy minimizes hoop displacement and prevents wrist flips.

THE CHALLENGE

Navigating the maze required precise spatial alignment and high positional stability. Several challenges had to be overcome:

  • • Robust detection of maze pose under variable lighting
  • • Calibration accuracy for mapping pixel → robot coordinates
  • • Avoiding IK branch flips in spatial trajectories
  • • Collision-free traversal in a tight environment
  • • Real-time pose adjustment and error compensation

THE SOLUTION

The final solution integrates perception, calibration, motion planning, and robot control into a unified execution pipeline:

  • • ChArUco-based intrinsic camera calibration
  • • ArUco marker detection for reliable maze localization
  • • Empirical hoop-plane calibration for pixel-to-world mapping
  • • Custom motion primitives for world-frame and tool-frame control
  • • IK-branch selection algorithm minimizing hoop displacement
  • • Variant-specific trajectory execution (A–D)

This architecture allowed the robot to adapt to changing maze placements and perform multiple trial runs with high repeatability.

RESULTS

The system achieved strong reliability across all evaluated variants. Variants A–C reached a 100% success rate, while Variant D achieved approximately 90% due to occasional less optimal IK configurations during orientation-based traversal.

The project demonstrated robust perception, stable control, and precise execution in a constrained 3D environment.
